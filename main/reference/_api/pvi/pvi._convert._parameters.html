class pvi._convert._parameters.Access(*args, **kwds)[source]#

Bases: enum.Enum

What access does the user have. One of:

  • R: Read only value that cannot be set. E.g. chipTemperature on a detector, or isHomed for a motor

  • W: Write only value that can be written to, but there is no current value. E.g. reboot on a detector, or overwriteCurrentPosition for a motor

  • RW: Read and Write value that can be written to and read back. E.g. acquireTime on a detector, or velocity of a motor
